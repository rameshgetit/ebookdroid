Стороннее ПО, использованное в этой программе:
  • mupdf легкий PDF/XPS просмотрщик, основанный на MuPDF.
  • djvu легкий DJVU просмотрщик, основанный на DjVuLibre
  • cbx легкий просмотрщик комиксов, основанный на libPng, libjpeg, GIFLIB, Unrar и zLib
  • Архиватор Snappy от Google
  • Клиентская библиотека Java CIFS
  • Библиотека Java CSS parser
Исходники mupdf и djvu могут быть найдены на странице загрузок проектного сайта.
