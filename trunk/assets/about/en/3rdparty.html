Third-party software used by this program:
  • mupdf a lightweight PDF and XPS viewer, based on MuPDF.
  • djvu a lightweight DJVU viewer based on DjVuLibre
  • cbx a lightweight Comicbook viewer based on libPng, libjpeg, GIFLIB, Unrar and zLib
  • Snappy compressor from Google
  • The Java CIFS Client Library
  • The Java CSS parser Library
mupdf and djvu sources can be found in Downloads section of the project site.
